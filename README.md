# Icon badge action

This action add a badge to your icons.

## Basic usage

```yaml
- name: Icon badge
  uses: damienaicheh/icon-badge-action@v1.0.0
  with:
    images-path: '/your-icon.png'
```

And you will get this banner on your icon:

![Icon badge](docs/icon-demo.png)
